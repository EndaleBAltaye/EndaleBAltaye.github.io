In [16]:
# draw dendogram 
plot(financesHClust,labels=Finances[,1],hang = -1,cex = 0.6,
     main='State clusters based on their Revenue and Expenditure',xlab="States",ylab="distance")
rect.hclust(financesHClust, k=5, border=1:5)#"red")

As we can see its clear that according to the Expenditure and Revenue values 5 cluster is a logical cluster to do. These are;

  • Cluster 1: Texas State
  • Cluster 2: New York State
  • Cluster 3: California State
  • Cluster 4: Pennsylvania, Illinois, Ohio, Michigan and Florida
  • Cluster 5: All states not listed above
